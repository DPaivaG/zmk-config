# ZMK firmware config for bluetooth chip nice!nano v2.0:
- kyria.keymap

## Description
It's a custom config for Kyria split ortho keyboard.

## Flashing flow
1. Install ZMK [1]
2. Save changes
2. Push to repo
3. Download flash u2 file after automatically built on GitHub Action
4. Reset the keyboard (splits)
